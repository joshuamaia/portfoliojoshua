Input - Colocar ZERO a esquerda

Exemplo de Input que coloca ZERO a esquerda. A validação fica a cargo do desenvolvedor.

Função para colocar o ZERO a esquerda: completaZeroEsquerda() {{ "{ const zeroesquerda: any = document.querySelector('#zeroesquerda'); const zeroesquerdaNumber = Number(zeroesquerda.value); zeroesquerda.value = String(zeroesquerdaNumber).padStart(6, '0'); }" }}

Código do Input: {{ 'Para que o código funcione, o input deve conter (keyup)="completaZeroEsquerda()" e id="zeroesquerda"' }}
