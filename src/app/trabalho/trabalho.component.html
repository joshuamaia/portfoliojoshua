CODATA - Setembro de 2014 até hoje
Mantém e evolui Sistemas de Informação utilizando a metodologia Scrum. Segue abaixo alguns dos projetos que faço parte:
  • SIOP PB (Sistema Integrado de Planejamento e Orçamento). Foram usadas as tecnologias Angular 9, Bootstrap 4, Html, Spring Boot, Query DSL, Java, Postgres, entre outros...
  • Transparência saúde. Sistema de prestação de contas das organizações sociais. No mesmo, temos lançamento de despesas, receitas, contratos, entre outros. Foram usadas tecnologias tais como JSF, Primefaces, Demoiselle, POI, Java, Eclipse, Critéria Builder, Postgres, entre outras.
  • STET (Sistema de prestação de contas). Foram usada as tecnologias Angular 9, Angular Material 9, Spring Boot 2, JWT, Hibernate Envers, Bootstrap 4, PrimeNg, Java dentre muitas outras tecnologias...
  • SIOP (Sistema Integrado de Planejamento e Orçamento). Trabalhando no desenvolvimento de relatórios com Jasper Studio.
  • Liquidação Client (Sistema que mostra dados do empenho, das liquidação e dos pagamentos relacionados a um determinado empenho). Foram usadas tecnologias como Angular 9, Angular Material 9, TypeScript, JavaScript, HTML, CSS, entre outras tecnologias...
  • Integração do Sistema de Procedimentos Policiais da Paraíba com o SINESP. Foram usadas tecnologias como Spring Boot, Spring Data JPA, Java, Eclipse, Postgres, Axis, entre outras
TOTVS - Agosto 2013 - Janeiro 2014
Trabalhando alocado no cliente UNICRED. As principais tecnologias utilizadas são Java, Jboss Seam, JSF e Richfaces. Banco de dados SQL Server.
DYNAVIDEO - Setembro 2009 - Fevereiro 2010
Fez parte da equipe do projeto no projeto ITVP. Dentre as principais atividades, desenvolvimento de 2 FileUploader, um para envio de um arquivo e outro para envio de múltiplos arquivos, ambos feitos em Swing, um mecanismo de importação de 2 bancos de dados onde foi utilizado JDOM, JDBC e JPA e refactoring de parte da API da empresa utilizando JPA. Nas atividades foi utilizado a IDE eclipse.
